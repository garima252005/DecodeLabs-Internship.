# AI Chatbot with Memory

## Project Overview
This project is a conversational AI chatbot that remembers previous messages during a live session.

## Features
- Gemini API Integration
- Conversation Memory
- Chat History Storage
- Memory Reset Command
- Terminal Interface

## Technologies Used
- Python
- Gemini API
- dotenv

## How to Run

1. Clone the repository
2. Create virtual environment
3. Install dependencies

pip install -r requirements.txt

4. Add Gemini API key in .env file

5. Run

python app.py

User: My name is Garima

User: What is my name?
